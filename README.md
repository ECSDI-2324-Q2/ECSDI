## Instruccions
> EXECUTAR TOTES LES COMANDES AL DIRECTORI ./shop/Agents

Per instalar les dependencies:
```sh
pip install -r requirements.txt
```

Per executar tots els agents:
```sh
./run.sh
```

Per parar tots els agents:
```sh
./stop.sh
```

Per executar un agent concret:
```sh
python NOMAGENT.py
```
